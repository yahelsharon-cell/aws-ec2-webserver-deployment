# AWS EC2 Web Project

## What is this?
A project demonstrating how to launch a virtual server (EC2) on AWS and turn it into a live web server.

## What I did:
1. **Launched an EC2 Instance:** Used Amazon Linux 2 (t2.micro).
2. **Security:** Configured the Firewall (Security Groups) to open Port 80 (HTTP).
3. **Linux Admin:** Connected via SSH and installed Apache Web Server (httpd).
4. **Deployment:** Hosted a custom "Hello Cloud" HTML page.

## Technologies
* AWS EC2 & Linux (Amazon Linux 2)
* Apache Web Server
* Bash Scripting
